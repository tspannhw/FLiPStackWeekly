### **Latest News from Apache NiFi**

We're kicking off this week's news with the official release of Apache NiFi 2.6.0. This new version brings several bug fixes and performance improvements to the data flow automation tool. You can find more details on the key changes and download the new version below.

* **Release Notes:** [Apache NiFi 2.6.0 Release Notes](https://issues.apache.org/jira/secure/ReleaseNote.jspa?projectId=12316020&version=12356071)  
* **Download:** [Apache NiFi Download Page](https://nifi.apache.org/download/)

### **Industry Updates: SQL, AI & Big Data**

This week saw a number of new developments and releases across the data and AI landscape, including updates from Snowflake and new open-source projects.

#### **SQL & Data Warehousing**

* **Snowflake Dynamic Tables:** A detailed look into Snowflake's Dynamic Tables, which simplify the creation of fresh, reliable data pipelines.  
  * [Read the article](https://select.dev/posts/snowflake-dynamic-tables)  
* **Postgres Streaming:** Check out pgstream, an open-source tool from Xataio for streaming changes from a PostgreSQL database.  
  * [View the GitHub repository](https://github.com/xataio/pgstream)  
* **OpenStreetMap to PostgreSQL:** The osm2pgsql project provides a flexible way to load OpenStreetMap data into a PostgreSQL database.  
  * [View the GitHub repository](https://github.com/osm2pgsql-dev/osm2pgsql)  
* **Snowflake Labs MCP:** A new framework from Snowflake Labs for building agentic applications on the Snowflake platform.  
  * [View the GitHub repository](https://github.com/Snowflake-Labs/mcp/tree/main)

#### **AI & Machine Learning**

* **Snowflake Cortex Pricing:** A breakdown of the pricing for Snowflake Cortex, their managed AI service for building LLM-powered applications.  
  * [Read the article](https://yukidata.com/blog/snowflake-cortex-pricing/)  
* **Snowflake Cortex Webinar:** A virtual hands-on lab from Snowflake on creating context-rich agents using Snowflake Intelligence and Cortex Knowledge Extensions.  
  * [Learn more about the webinar](https://www.snowflake.com/webinars/virtual-hands-on-labs/virtual-hands-on-lab-create-context-rich-agents-using-snowflake-intelligence-and-cortex-knowledge-extensions-2025-10-01/)  
* **Google Agentic Commerce:** A look at a new project from Google focused on agentic systems for e-commerce.  
  * [View the GitHub repository](https://github.com/google-agentic-commerce/AP2)  
* **semlib:** An open-source Python library for building semantic models.  
  * [View the GitHub repository](https://github.com/anishathalye/semlib)
