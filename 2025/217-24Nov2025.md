## **üöÄ Snowflake Platform & Infrastructure Updates**

Significant advancements have been made to Snowflake's core services, enhancing performance, governance, and developer productivity.

* **Next-Gen Snowpipe Streaming:** The architecture has been updated for Snowpipe Streaming, delivering a new high-performance data ingestion experience.  
  * [Deep Dive on New Architecture](https://medium.com/snowflake/snowpipe-streaming-with-the-new-high-performance-architecture-f59b6f22ef7b)  
  * [Engineering Blog Post](https://www.snowflake.com/en/engineering-blog/next-gen-snowpipe-streaming-architecture/)  
* **Snowpark Container Services (SPCS) Enhancements:** New features include block storage volume encryption support and updates to image repository creation.  
  * [Encryption Support for Block Storage](https://docs.snowflake.com/en/developer-guide/snowpark-container-services/block-storage-volume#encryption-support)  
  * [Image Repository SQL Reference](https://docs.snowflake.com/en/sql-reference/sql/create-image-repository#optional-parameters)  
* **Storage Lifecycle Policies (GA):** These policies are now Generally Available, offering simplified storage management and optimized cost control.  
  * [Official Blog Post](https://www.snowflake.com/en/blog/storage-lifecycle-policies-ga/)  
  * [User Guide Documentation](https://docs.snowflake.com/user-guide/storage-management/storage-lifecycle-policies)  
* **Iceberg Table Innovations:** New guides cover declarative pipelines using Dynamic Iceberg Tables and querying Iceberg tables via external engines using Snowflake Horizon.  
  * [Declarative Pipelines Guide](https://www.snowflake.com/en/developers/guides/declarative-pipelines-using-dynamic-iceberg-tables/)  
  * [External Query Engine via Horizon](https://docs.snowflake.com/en/user-guide/tables-iceberg-query-using-external-query-engine-snowflake-horizon)  
* **Terraform Provider Update:** Version v2.11.0 has been released for the Snowflake Terraform provider.  
  * [Release Notes](https://github.com/snowflakedb/terraform-provider-snowflake/releases/tag/v2.11.0)
